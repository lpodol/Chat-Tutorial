Git and Github

Git - Version Control

Git is a version control system, a better analogy would be that it a bunch of snapshots(the commits) of your code. You are able to see the path in the order of which they are created, as well as experiment with various branches to experiment with various versions.


Github - Collaboration

Github is a webpage for you to publish your Git in repositories. You are able to collaborate with various people on Github.


Git On Github

Start by going to Github and creating an account. If you have trouble creating a free account, please hold down the power button on the computer until blank and proceed to call your ISP to cancel your internet service subscription.


SSH key to start

How do we know which computer is safe to push and save? This is where the SSH key comes in. You will need to this to identify trusted computers to push into your repositories. After you create your free Github account, please read Github SSH key to setup your account and computer.


Using Github

Now that you've validated your computer with Github, you can start pushing your work on to github! Please read Push to Github. You can start by pushing your very own ChatApp onto Github!
