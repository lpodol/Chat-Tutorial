Host Live on Digital Ocean


When you're ready to actually launch the site onto the web after the tests, you can do it through Digital Ocean. Similar to hosting on a local server through the terminal, you will Secure Shell into digital ocean. Secure Shell or "SSH" is the protocol to cryptographic network protocol for secure data communication, remote command-line login, remote command execution, and other secure network services between two networked computers.

Start pushing everything to github then by going to your terminal, you will then type in:

ssh root@YOURNAME.princesspeach.nyc


Once you are connected, proceed to clone all the items you pushed to git in regards to your ChatApp by typing in:



git clone


This must be done while inside the root of your Digital Ocean Server. Proceed into the folder containing your ChatApp. You must then install the node package manager by typing in npm install install onto the Digital Ocean server.

To start your server on Digital Ocean type in node servername.js. This will turn your Digital Ocean box into a server.

Then open another terminal and like before you must ssh into Digital Ocean, when you connect to the root run http-server so you may access your ChatApp live at http://thename.princesspeach.nyc.
