Welcome to the exciting world of making a ChatApp.
You will learn how to build a simple ChatApp from
this awesome tutorial.

This tutorial has following objectives:
1. Get set up with "npm" and "websockets".
2. Write code for the client and server.
3. Style the ChatApp.
4. Set up a Git/Github repo.
5. Test your work locally on your computer.
6. Run your first ChatApp by hosting on a server.
