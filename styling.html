Styling your webpage

Here is an example of how to start styling your ChatApp. We will begin with your HTML file.

Take an element from your HTML file and start styling your page with CSS.
We will use below CSS sheet as an example:

margin: auto;
{margin-top: 25px;}

Setting the width of a block-level element will prevent it from stretching out to the edges of
its container to the left and right. You can set the left and right margins to auto to horizontally
center that element within its container. The element will take up the width you specify, then the
remaining space will be split evenly between the two margins.
Also, the top, right, bottom, and left margin can be changed independently using separate properties.
The only problem occurs when the browser window is narrower than the width of your element. The browser resolves this by creating a horizontal scrollbar on the page.

percent width
{width: 60%;}

Percent is a measurement unit relative to the containing block. It's great for images: here we make an image that is always 50% the width of its container. You can use percent for layout, but this can require more work.

float
{float: right;}

With CSS float, an element can be pushed to the left or right, allowing other elements to wrap around it. Float is often used with images, but it is also useful when working with layouts.
